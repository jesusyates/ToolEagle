## ToolEagle

**ToolEagle** is a simple Next.js site that offers free tools for creators.

The first tool is a **TikTok Caption Generator** that turns a short video idea into a ready‑to‑use caption with emojis and hashtags.

### Tech stack

- **Next.js 14** with the App Router
- **TypeScript** + **ESLint**

### Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser to view ToolEagle.
