## Publish-ready package framing (what to paste)

### Hook
Stop using generic challenge hashtags that get lost in the noise.

### Script beats
- Open with a relatable struggle about hashtags not working
- Show a before/after example of niche vs. generic tags
- Break down how the Instagram algorithm favors specificity
- Demonstrate how to mix challenge, niche, and community tags
- End with a clear action to test one set immediately

### Caption / title
