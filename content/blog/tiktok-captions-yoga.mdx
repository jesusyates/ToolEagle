## Publish-ready package framing (what to paste)

### Hook
Struggling to write captions that get saves for your yoga videos?

### Script beats
- List 3 common yoga caption mistakes that hurt engagement.
- Show a before-and-after example of a weak vs. strong caption.
- Reveal the one element all high-performing yoga captions share.
- Present this list of 30 ready-to-use captions as a solution.

### Caption / title
30 Yoga Captions That Actually Get Saves (Steal These)

### CTA
Which caption will you try first? Comment below!

### Hashtags
#YogaTikTok #YogaCaption #MindfulMovement #YogaForBeginners #WellnessTok #YogaTips #YogaCommunity #TikTokYoga #YogaPractice #CaptionsThatConvert #ContentCreatorTips #ShortFormContent

### Why it works
- Addresses specific pain points (tightness, alignment, mindset) making it highly relatable.
- Uses advisory language ('can help', 'might be') which builds trust without overpromising.
- Each caption is a mini-lesson, providing value beyond just describing the video.
