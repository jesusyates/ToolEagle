# V104.2 — Douyin Conversion Hardening

**Scope**: `/zh/douyin` and the three Douyin generators only. English/global site unchanged as primary; 小红书/快手 remain non-public.

## Paywall (stronger free vs Pro)

- `clientRoute` matching `/zh/douyin-{caption,hook,script}-generator` triggers **`douyinHardPaywall`** in `POST /api/generate-package`.
- `toFreeVisiblePackages`: **1** visible package (plus supporter extra slots), tighter caps (`CAP_*_DOUYIN`).
- `buildLockedPreviews(..., { douyinRichTeasers })`: real **口播截断** + `structureHint` from model `why_it_works` (truncated).
- UI: `PostPackageResults` + `PostPackageToolClient` with `douyinConversionMode`; locked cards show **非空**口播预览条。

## Pricing / package positioning

- Default **`getCnProMonthlyCny()`** fallback **¥29** (override with `CN_PRO_MONTHLY_CNY` / `NEXT_PUBLIC_CN_PRO_MONTHLY_CNY`).
- `/zh/pricing`: copy emphasizes **AI 算力次数、更完整结果、高频创作、持续做内容**; Pro card **最受欢迎** + **大多数创作者选择** + `id="cn-pro-highlight"`.

## Publish-ready signals

- Douyin result tags (ZH): 带货 / 干货 / 情绪共鸣 / 个人IP / 引发评论 / 完播导向 / 知识口播 / 娱乐节奏.
- Pro: section labels **为什么这个开头能抓注意力** / **为什么这个结构适合口播短内容** (mapping `why_it_works` / `posting_tips`).
