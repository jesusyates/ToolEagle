# V109.3 — 英文首页「全球创作者工具入口」重构

**目标**：清晰主路径、去除中文干扰、不改 layout/header/footer/路由与 SEO 顶层机制。

---

## 1. 新模块结构（自上而下）

| 区块 | 组件 | 说明 |
|------|------|------|
| 首屏 Hero | `HomeHeroGenerate` | H1 + 副标题 + **唯一主 CTA** + 次 CTA + 信任文案 + Pro 链（弱化） |
| 主工具入口 | `HomeQuickToolsSection` | 「Start creating in seconds」+ 5 张卡片（单列→多列） |
| 示例输出 | `HomeExampleOutputs` | 「See what you can create」+ 2 组静态 hook/caption/script 示例 |
| 信任 | `ValueProofBlock` | 保留，上移紧随示例后 |
| 指南（仅 EN） | `TrendingMakeMoneySection` `englishOnly` | 仅 `/en/how-to/*`，无 `/zh`、无「中文指南」标签 |
| SEO / 发现（下移） | 原 `HomePageClient` 下部 | Platform tools、Popular tools、Why、Blog、`CreatorModeDemo` |

---

## 2. 主 CTA 指向

- **主按钮**：「Start with AI Caption Generator」→ **`/tools/ai-caption-generator`**（全包 / Post Package 主工具，与卡片区「Post Package Generator」一致）。
- **次按钮**：「Browse all tools」→ **`/tools`**。

---

## 3. 移除或不再出现在英文首页主体的内容

| 项目 | 说明 |
|------|------|
| 原 Hero 内大段 **textarea + Generate** | 与「单一主路径」冲突，已移除 |
| **`TrendingMakeMoneySection` 使用 `locale: zh`** | 曾混入中文指南与 `/zh` 链接；现改为 **`englishOnly`**，仅 EN how-to |
| **TopResultsShowcase** | 首页不再挂载（避免与 `HomeExampleOutputs` 重复）；文件仍保留供它用 |
| 中部重复模块 | 已移除：Growth Kit 双按钮条、「Popular for creators」药丸、三列 Create viral / Learn AI / Creator inspiration 大卡片区 |

**说明**：`ZhHomepageKeywordSections` 仅用于 **`/zh` 中文首页**，从未挂在英文 `/`，无需改动。
