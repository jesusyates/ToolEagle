# V109.6 — English site header (site-level fixes)

**Scope**: English top bar only (no layout/routes/metadata/sitemap changes). Converge on a clearer global “portal” header: Home entry, single pricing CTA, English-only market dropdown copy, light density.

## 1. Header structure — before vs after

| Area | Before | After |
|------|--------|--------|
| Brand | Logo + slogan, linked to `/` | Same, plus `aria-label` from `nav.homeLogoAria` and focus ring for clearer “home” affordance |
| Primary nav | Tools, Guides, Pricing (+ optional Pro pill next to nav in older builds) | **Home**, Tools, Guides, Pricing |
| Pricing vs Pro | Two top-level paths could both imply “pay / upgrade” | **Pricing only** in the strip; no `UpgradeLink` (“Pro — Grow faster”) in `SiteHeader` |
| Market | Dropdown / label sometimes mixed zh/en | Dropdown label + options from `nav.*` (English messages on EN site) |
| Right cluster | Nav + Pro + market + auth | Nav + market + auth (lighter) |

## 2. Home entry — how it works

- **Explicit nav**: First item in `primaryNav` is `{ href: "/", key: "home" }` → label **Home** (`messages/en.json` → `nav.home`).
- **Brand block**: `TranslateAwareLink` to `/` with `aria-label={t("homeLogoAria")}` (“ToolEagle — home”) so screen readers and keyboard users get a clear home action from the logo/title stack.

**File**: `src/app/_components/SiteHeader.tsx`.

## 3. “Pro — Grow faster” removed?

**Yes** from the English header. `SiteHeader` no longer renders `UpgradeLink` / `nav.upgrade`. Users reach paid flows via **Pricing** (`/pricing`) and other in-app surfaces as before.

`nav.upgrade` remains in `messages/en.json` for any other consumer; not used in this header.

## 4. Market / language selector — visible copy (EN dropdown)

All user-visible strings for `presentation="dropdown"` come from `useTranslations("nav")` in `MarketSelector`.

| Key | English copy |
|-----|----------------|
| `marketSelectorLabel` | Region |
| `marketSelectorAria` | Choose region or language |
| `marketOptionEn` | English (Global) |
| `marketOptionZh` | Simplified Chinese (China) |
| `marketOptionEs` | Spanish |

**File**: `src/components/locale/MarketSelector.tsx` (dropdown branch only). Segmented mode on the Chinese header is unchanged (e.g. “英文主站”).

**Parity**: `messages/zh.json` includes the same keys so missing-key fallbacks are avoided if locale bundles differ.

## 5. Right-side density — light tweaks

- Removed the extra Pro pill from the header row.
- Container uses `flex-col` / `gap-3` on small screens, `sm:flex-row` + `min-w` guard on the right block so nav, region select, and auth don’t crush together.

## 6. Brand / slogan

- `nav.slogan` (EN): **“Global AI tools for creators”** — short, portal-oriented line under the wordmark.
